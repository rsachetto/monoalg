About MonoAlg3D

MonoAlg3D is a high-performance program for solving the 3D monodomain equation using the Finite Volume Method. It's designed for cardiac electrophysiology simulations with GPU acceleration and supports multiple output formats including VTK for visualization in ParaView.

Getting Started

Prerequisites

  • Linux-based operating system
  • NVIDIA Driver
  • CUDA library

Compile

$ ./build.sh

The binary files will be saved in the bin folder.

Running Examples

$ bin/MonoAlg3D -c example_configs/cuboid_ohara.ini

The output will be saved in VTK format for visualization in ParaView.
